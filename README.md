# IEP Process Tutorial

An interactive self-learning tutorial covering the complete IEP process under federal IDEA law.

## What It Covers
- Child Find and student identification
- Referral process and timelines
- Consent to evaluate (60-day clock)
- Multi-disciplinary evaluation
- Eligibility determination (13 disability categories)
- IEP meeting and team members
- Placement and Least Restrictive Environment
- IEP consent and mandated services
- Annual review and reevaluation
- Dispute resolution (4 options)

## Features
- Choose your role: Parent, Special Ed Teacher, CSE Coordinator, or General Ed Teacher
- Follow fictional student "Alex" through every step
- Interactive decision branches — real scenarios with instant feedback
- Knowledge check quizzes after each step
- Progress tracking and completion certificate
- Based entirely on federal IDEA law — applies in all 50 US states

## Deployment
Deployed on Streamlit Community Cloud.

**Main file:** `Home.py`

## Requirements
- Python 3.9+
- streamlit >= 1.32.0

## Running Locally
```bash
pip install -r requirements.txt
streamlit run Home.py
```
